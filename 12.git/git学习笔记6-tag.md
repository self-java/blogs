## 什么是标签

tag就是一个让人容易记住的有意义的名字，它跟某个commit绑在一起。标签没什么特别的，就是标记而已。

## 如何打标签
在本地打标签，命令如下：
```
git tag xxx
```
这样就打了一个叫xxx的标签。



上面这个是简单的打标签的方法，还有一个复杂点的方法。
```
git tag -a yyy -m "this is version yyy"
```
打标签还有好多用法，比如在某一个commit上打标签，加PGP等，用的不多，我也不会。

## 怎么查看已有的标签
下面这个命令查看标签
```
git tag
```

用下面这个命令查看某一个标签的信息

```
git show xxx
```

## 如何删除标签？
删除标签的命令如下
```
git tag -d xxx
```
## 怎么切换到某一个标签上
